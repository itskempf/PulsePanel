# PulsePanel

PulsePanel is a **Windows-native, provenance-first, accessibility-driven** server management panel — built to be the definitive open-source alternative to tools like WindowsGSM, but with our own DNA:

- **Blueprint-driven**: Game-agnostic install/config/update definitions with full provenance logging.
- **Accessibility-first**: Dyslexia-friendly typography, high-contrast modes, keyboard-first navigation.
- **No paywalls, no feature gating**: Every capability unlocked for everyone.
- **Compliance guardrails**: License checks before blueprint/plugin execution.
- **Native performance**: Built with WinUI 3 for a lean, resource-efficient experience.

## Key Features
- Multi-server management with deterministic automation pipelines.
- One-click install/import/update with acceptance criteria before execution.
- Provenance logging for every action — exportable for audits.
- Blueprint Catalog with attribution lock-in and non-commercial license enforcement.
- Extensible via dual-layer architecture: blueprints for game logic, plugins for UI/automation.
